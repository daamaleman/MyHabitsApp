## 📱 Sobre el Proyecto

**HabitFlow** es una aplicación de seguimiento de hábitos diseñada bajo una filosofía de **minimalismo moderno**. El objetivo es proporcionar una herramienta poderosa pero visualmente tranquila, que ayude a los usuarios a mantener la constancia en sus rutinas diarias mediante una interfaz intuitiva y animaciones fluidas.

## 📐 Arquitectura

El proyecto sigue el patrón **MVVM (Model-View-ViewModel)**, asegurando una separación clara de responsabilidades y una UI reactiva que responde instantáneamente a los cambios en el estado de los datos.

1.  **UI Layer:** Jetpack Compose para componentes declarativos y modernos.
2.  **Navigation:** Compose Navigation con transiciones personalizadas y gestión de backstack.
3.  **Data Layer:** Gestión de persistencia mediante `SharedPreferences` y `kotlinx.serialization`.

---

## 🚀 Instalación y Uso

1.  Clona este repositorio.
2.  Abre el proyecto en **Android Studio Ladybug (2024.2.1)** o superior.
3.  Sincroniza el proyecto con los archivos de Gradle.
4.  Ejecuta la aplicación en un dispositivo físico o emulador con **API 26 (Android 8.0)** o superior.
